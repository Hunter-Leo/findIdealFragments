## Install from source

```bash
uv sync
uv build
uv pip install -e .
```

## Install from remote
```bash
uv tool install git+https://github.com/Hunter-Leo/findIdealFragments
uv tool upgrade find-ideal-segments # upgrade tool
```


## Run

### Method 1
```bash
gccontent --help
```

### Method 2
```bash
uvx --from find-ideal-segments gccontent
```
